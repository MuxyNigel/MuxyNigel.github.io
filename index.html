Pungwe Valley Grassroots Football League Management System (Live)

A full-stack web-based Management Information System (MIS) designed to support grassroots football administration in Honde Valley, Zimbabwe. The system improves transparency, integrity, and accessibility of league information by digitizing fixtures, results, standings, and team data in line with ZIFA’s grassroots football development vision.

This solution operates as a grassroots football MIS, emphasizing system design, data flows, and process standardization at community level. It models core football operations—fixtures, results, standings, and administration—into a centralized system, improving data integrity, traceability, and reporting.

Live System: https://pungwevalleyleaguemanagement.onrender.com

Objectives:

  • Digitize grassroots football league management for rural communities.
  • Improve transparency and integrity in match results and league standings.
  • Provide a centralized platform for fixtures, results, and team information.
  • Support ZIFA’s vision of structured grassroots football development.
  • Enable public access to verified football information across Zimbabwe.

Key Features:

  • Match scheduling and result tracking
  • Automatic league table and standings calculation
  • Public-facing league portal
  • Secure Django Admin interface for league administrators
  • Cloud-hosted PostgreSQL database

Tools & Technologies:

  • Django (Python)
  • PostgreSQL
  • HTML, CSS, Bootstrap
  • Gunicorn
  • Render Cloud Platform
  • Git & GitHub (CI/CD)

Experience & Skills Gained:

  • Designing and deploying a production-ready Django application
  • Database modeling for real-world sports management systems
  • PostgreSQL integration in cloud environments
  • Deployment and debugging on Render (free-tier constraints)
  • Admin panel customization and data governance
  • Building scalable MIS architectures aligned with national governance structures

Impact: This project demonstrates how technology can strengthen grassroots football governance, enhance visibility of rural leagues such as Honde Valley, and contribute to transparent football development across Zimbabwe.

AgriAssistant (Final Year Project)

AgriAssistant is an AI-powered agricultural support system designed to assist farmers in making data-driven decisions. It integrates land suitability analysis, crop yield forecasting, plant disease detection, interactive data visualization, and an intelligent chatbot to provide comprehensive support for sustainable farming.

Objectives:

  • Land Suitability Analysis: Analyze soil and environmental data to recommend optimal crops for specific land parcels, enhancing yield and resource efficiency.
  • Crop Yield Prediction: Forecast potential crop yields based on land and environmental conditions to aid in planning and resource management.
  • Plant Disease Detection: Utilize CNN-based image classification techniques to identify plant diseases early, enabling timely interventions.
  • Data Visualization: Visualize crop yield trends interactively using user-provided CSV datasets for regional and temporal analysis.
  • Farmer Support Chatbot: Provide instant guidance and answers to agricultural queries, empowering farmers with accessible expert advice.

Skills Gained:

  • Data analysis & interpretation
  • Machine learning and deep learning (CNNs)
  • Environmental and soil data analysis
  • Natural language processing & chatbot development
  • Database management (SQL)
  • AI model deployment in real-world applications

City of Mutare Billing Analytics Dashboard

This Data App project is a comprehensive web-based Billing Analytics Dashboard designed to revolutionize how the City of Mutare analyzes and visualizes billing-related data across all services, including rates, sewer, water, street lights, and more. It aims to centralize data, improve reporting, and standardize KPIs for better decision-making.

Tools: R, GIS libraries (Mapview, Folium), PostgreSQL, Shiny
