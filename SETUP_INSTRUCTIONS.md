## ❓ Common Questions

**Q: Will my GitHub Pages URL change?**  
A: No! GitHub Pages will serve the updated `index.html` at the same URL.

**Q: Do I need to install anything?**  
A: No! The portfolio uses pure HTML5, CSS3, and vanilla JavaScript - no dependencies.

**Q: How do I add new projects?**  
A: 
1. Create new folder in `assets/images/projects/[project-name]/`
2. Add your images
3. Copy a project card in HTML and modify
4. Update paths: `assets/images/projects/[project-name]/[image].png`

**Q: Can I change colors?**  
A: Yes! Look for the CSS `:root` variables section and update colors.

**Q: Will it work on all browsers?**  
A: Yes! Tested on Chrome, Firefox, Safari, and mobile browsers.
